The Fleet Rome Variant

Last revised: October 13, 1992

Send comments or corrections (only for the HTML version of this file) to Doug Massey (masseyd@valhalla.btv.ibm.com).

You can receive a copy of this file via E-mail by sending your local Judge a message with "get info.fleet_rome" in the body. 

The "Fleet Rome" variant differ from regular Diplomacy in only one
detail;  Italy starts with a Fleet in Rome instead of an Army.

To sign onto a fleet_rome game, the fleet_rome keyword must be specified
as part of the variant identification:

    signon ?name password fleet_rome

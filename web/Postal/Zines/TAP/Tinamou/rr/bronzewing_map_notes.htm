Terrain Build Costs
Plains, Towns: 1 (basic build cost)
Mountain: +2 per half-hex
River: +2 per hex-side

Actual Build Costs
1: Plains - Plains
3: Plains - River - Plains; Plains - Mountain
5: Mountain - Mountain; Plains - River - Mountain

Terrain Notes
The Lake at Q34 is impassable.
Aozou and Feda are in the hills and cost 3 to build into.
Players can only build one hex into adjacent countries for specials except for through F36, F37, F38 and B43 to access Nigeria.

Start Positions
Brisbane, Maryborough, Bundaberg, Rockhampton, Mackay, Townsville, Cairns, Cooktown, Normanton
More than one player may start at each location.

Specials

BASIC:
1:   Northern Territory: Connect to Camooweal (hex P2 labelled 'NT')
2-4: Port: Cairns, Townsville, Bowen, Mackay, Gladstone, Maryborough or Brisbane.
      (Optional rule: in postal games, Maryborough is available only in rounds 7-9)
5-6: NSW: Connect to any hex containing NSW State Rail Track (inc. R89 & S87 in Qld)

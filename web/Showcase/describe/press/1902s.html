Broadcast from Master:

    The player for whom I extended the deadline has resolved the
    access problem.  Since the deadline was only extended because
    of those problems, will the player(s) who have set wait please
    set nowait?
    
    Also, one of the players has expressed concern at the rate at
    which this game is moving because he may be losing access in
    mid-april.  I think that some delays are unavoidable for one
    reason or another, but I hope that the game can proceed more
    quickly now that the long holidays are over.
    
    If everyone would like the game to move more quickly, I can
    change the deadlines for move phases from 72 hours to 48 hours.
    However, since some people may feel like this is too short for
    a game with press, so I will only make this change if every
    player agrees.  Please let me know your preference.
    
                - Simon
